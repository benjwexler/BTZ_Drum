B*T*Z Drum is a virtual drum machine with sounds from hip-hop's biggest producers

<h3> Features: </h3>
<li> Note Repeat </li>
<li> Midi Support </li>
<li> Custom Key Mapping </li>
<li> Multiple Kits </li>

<h3> Built with: </h3>

<li> Javascript </li>
<li> Web Audio API </li>
<li> Bootstrap </li>
